# 🌸 旅遊規劃行程計畫表 🌸

一個互動式的 **旅遊規劃行程計畫表**，可以自訂每日行程、口袋名單、代買清單，並透過瀏覽器直接管理和分享行程。

---

## 功能特色

- 📅 **多日行程規劃**  
  自由新增/刪除天數，每天可編輯景點名稱、時間、描述。

- 📝 **口袋名單管理**  
  收藏想去但尚未排入行程的餐廳或景點，隨時加入每日行程。

- 🛍️ **代買清單**  
  每個景點可記錄代買項目，支援勾選完成與刪除。

- 💾 **匯出/匯入 JSON**  
  可以備份或恢復行程資料，方便跨裝置使用。

- 📋 **分享功能**  
  生成可複製連結，讓朋友也能快速查看行程。

- 🎨 **美觀 UI**  
  日夜間漸層配色、卡片式設計，操作直覺又可愛。
